PyMOLの簡単インストール(2017.10.08　更新)

  1. Download PyMOL 2.0のDOWNLOAD NOWをクリックし
  2. Windows EXE Installerをクリックし、ファイルをダウンロード
  3. ダウンロードしたファイルを実行
  4. 実行ファイルはインストール下フォルダーのPyMOLWin
  5. 起動時にライセンスのActivation画面が出てくるが、これをSkip Activation


MacPyMOL(2018.08.01　更新)

    Macの方も同様のようです。 Macの方はコチラからどうぞ。Windows同様簡単になったようです。












PyMOLインストール(2016.10.14　更新)

  1. PythonダウンロードページのLoking for a specific release?の下のPython2.7.11(2.7.12にはpipが同梱されていない）をクリックし、Windows x84-64 MSI installerをダウンロード
  2. ダウンロードしたファイルを実行
  3. Unofficial Windows Binaries for Python Extension Packagesから

    numpy-1.11.2+mkl-cp27-cp27m-win_amd64.whl
    Pmw-2.0.1-py2-none-any.whl
    pymol-1.8.4.0-cp27-cp27m-win_amd64.whl
    pymol_launcher-1.0-cp27-cp27m-win_amd64.whl

    をC:\Python27へダウンロード

  4. コマンド プロンプトを開き、cd C:\Python27

    5. Scripts\pip install wheel
    python -m pip install --upgrade pip

    これでpipが7.1.2から8.1.2にアップデートされる

    Scripts\pip install Pmw-2.0.1-py2-none-any.whl
    Scripts\pip install numpy-1.11.2+mkl-cp27-cp27m-win_amd64.wh
    Scripts\pip install pymol-1.8.4.0-cp27-cp27m-win_amd64.whl
    Scripts\pip install pymol_launcher-1.0-cp27-cp27m-win_amd64.wh


    PyMOL実行ファイルは、C:\Python27\PyMOL.exeです。
    PyMOL.exeのショートカットを作成しておくと便利です。
    (右クリック→送る→デスクトップ(ショートカット作成))
