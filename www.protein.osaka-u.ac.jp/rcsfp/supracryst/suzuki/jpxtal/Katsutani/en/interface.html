Download script

By using the script called "InterfaceResidues", you can select interface residues.
  1. Access PyMOL wiki InterfaceResidues
  2. Make a copy of in the gray box under the The code, and then paste it into a text file
  3. Save as "InterfaceResidues.py"
Or download InterfaceResidues.py


Run color_h on PyMOL

  1. Open PDB file
  2. File → Run → InterfaceResidues.py
  3. Enter the comman line
    4. interfaceResidue 3ALP, chain A, chain B
    3ALP → name of object
    chain A, chain B → chain ID of two chain


  4. Object panel (interface) is added.



Show interface residues

Add object panel of chainA and chainB, and change "color" and "show".
select molA, chain A
select molB, chain B
