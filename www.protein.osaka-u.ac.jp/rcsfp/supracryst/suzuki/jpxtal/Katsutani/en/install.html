Win PyMOL

  1. Python download page
    Download Python2.7.5 Windows installer
  2. Run python-2.7.5.msi
  3. Unofficial Windows Binaries for Python Extension Packages
    Download "pymol-1.6.0.0.win32-py2.7.exe"
  4. Run "pymol-1.6.0.0.win32-py2.7.exe"
  5. Install under Python Install directory (C:\Python27\PyMOL)
PyMOL executable file is C:/Python/PyMOL.exe. It is convinient to create a shortcut for the PyMOL.exe.
